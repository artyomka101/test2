# Образовательный портал - EduPortal

Информационная система для управления образовательными курсами, студентами и преподавателями.

## Описание проекта

Система разработана на языке C# с использованием Windows Forms и базы данных PostgreSQL. Позволяет:

- Управлять курсами (добавление, редактирование, удаление, поиск)
- Управлять студентами (добавление, редактирование, удаление)
- Управлять преподавателями (добавление, редактирование, удаление)
- Просматривать данные в удобном табличном виде

## Технологии

- **Язык программирования**: C# (.NET 6.0)
- **GUI Framework**: Windows Forms
- **База данных**: PostgreSQL
- **ORM**: Npgsql (драйвер для PostgreSQL)
- **IDE**: Visual Studio

## Структура базы данных

### Таблицы:
1. **Teachers** - преподаватели
   - teacher_id (PK)
   - first_name, last_name
   - subject, email, phone

2. **Courses** - курсы
   - course_id (PK)
   - course_name, duration
   - teacher_id (FK)
   - credits, semester, description

3. **Students** - студенты
   - student_id (PK)
   - first_name, last_name
   - email, group_name
   - enrollment_date

4. **Student_Courses** - связь студентов и курсов
   - id (PK)
   - student_id (FK), course_id (FK)
   - enrollment_date, grade

## Установка и настройка

### Предварительные требования:
1. Visual Studio 2022 (Community или выше)
2. PostgreSQL 12+ с pgAdmin
3. .NET 6.0 Windows Desktop Runtime

### Шаги установки:

1. **Настройка базы данных:**
   ```sql
   -- В pgAdmin создайте базу данных
   CREATE DATABASE "EduPortal";
   
   -- Выполните скрипт database_setup.sql
   ```

2. **Настройка подключения:**
   - Откройте файл `EduPortal/Data/DatabaseManager.cs`
   - Измените строку подключения:
   ```csharp
   _connectionString = "Host=localhost;Port=5432;Database=EduPortal;Username=postgres;Password=your_password";
   ```

3. **Сборка проекта:**
   ```bash
   dotnet build
   ```

4. **Запуск:**
   ```bash
   dotnet run
   ```
   Или нажмите F5 в Visual Studio

## Функциональность

### Вкладка "Курсы"
- Просмотр всех курсов в таблице
- Добавление нового курса
- Редактирование выбранного курса
- Удаление курса
- Поиск курсов по названию
- Выбор преподавателя из списка

### Вкладка "Студенты"
- Просмотр всех студентов
- Добавление нового студента
- Редактирование данных студента
- Удаление студента

### Вкладка "Преподаватели"
- Просмотр всех преподавателей
- Добавление нового преподавателя
- Редактирование данных преподавателя
- Удаление преподавателя

## Использование

1. **Добавление данных:**
   - Заполните поля в нижней части формы
   - Нажмите кнопку "Добавить"

2. **Редактирование:**
   - Выберите строку в таблице
   - Данные автоматически загрузятся в поля
   - Измените нужные поля
   - Нажмите "Редактировать"

3. **Удаление:**
   - Выберите строку в таблице
   - Нажмите "Удалить"
   - Подтвердите действие

4. **Поиск курсов:**
   - Введите название курса в поле поиска
   - Нажмите "Найти"

## Примеры SQL-запросов

### Получение всех курсов с преподавателями:
```sql
SELECT c.course_id, c.course_name, c.duration, c.credits, c.semester, c.description,
       t.first_name || ' ' || t.last_name as teacher_name
FROM Courses c
JOIN Teachers t ON c.teacher_id = t.teacher_id
ORDER BY c.semester, c.course_name;
```

### Поиск курсов по названию:
```sql
SELECT * FROM Courses 
WHERE LOWER(course_name) LIKE LOWER('%программирование%');
```

### Добавление нового курса:
```sql
INSERT INTO Courses (course_name, duration, teacher_id, credits, semester, description) 
VALUES ('Новый курс', 80, 1, 3, 1, 'Описание курса');
```
