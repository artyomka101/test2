Система учёта учебных достижений

Введение

Система учёта учебных достижений представляет собой консольное приложение для управления оценками студентов. Программа позволяет регистрировать пользователей, добавлять и редактировать оценки, рассчитывать средние баллы и осуществлять поиск по различным критериям.

Системные требования

  • Операционная система: Windows, Linux, macOS
  • Python версии 3.6 или выше
  • Свободное место на диске: 10 МБ

Установка и запуск

  1. Убедитесь, что Python установлен на вашем компьютере
  2. Скачайте все файлы проекта в одну папку
  3. Откройте командную строку в папке с проектом
  4. Выполните команду: python main.py
Важно: При первом запуске система автоматически создаст базу данных и заполнит её тестовыми данными.

Структура главного меню

После запуска программы отображается главное меню с двумя режимами работы:

Для неавторизованных пользователей:

  • 1. Войти в систему
  • 2. Регистрация
  • 0. Выход

Для авторизованных пользователей:

  • 1. Просмотр оценок
  • 2. Добавить оценку
  • 3. Редактировать оценку
  • 4. Удалить оценку
  • 5. Статистика
  • 6. Поиск по предметам
  • 7. Выйти из аккаунта
  • 0. Выход из программы

Работа с системой

Регистрация нового пользователя

  1. В главном меню выберите пункт "2. Регистрация"
  2. Введите уникальный логин (если логин уже существует, система сообщит об ошибке)
  3. Введите пароль
  4. Введите полное имя (ФИО)
  5. Введите название группы
  6. При успешной регистрации появится сообщение о завершении процедуры

Вход в систему

  1. Выберите пункт "1. Войти в систему"
  2. Введите логин и пароль
  3. При правильных данных откроется персональное меню

Просмотр оценок

Функция отображает все оценки текущего пользователя в табличном формате с указанием:

  • ID оценки
  • Название предмета
  • Полученная оценка
  • Тип оценки (экзамен, зачёт, домашнее задание)
  • Дата получения

В конце списка отображается общий средний балл студента.

Добавление новой оценки

  1. Система отобразит список доступных предметов с их ID
  2. Введите ID выбранного предмета
  3. Введите оценку от 2 до 5
  4. Выберите тип оценки:
    • 1 - Экзамен
    • 2 - Зачёт
    • 3 - Домашнее задание
  5. Оценка будет сохранена с текущей датой

Редактирование оценки

  1. Система покажет все ваши оценки с их ID
  2. Введите ID оценки, которую хотите изменить
  3. Введите новую оценку от 2 до 5
  4. Изменения сохранятся автоматически

Удаление оценки

  1. Выберите оценку из списка по её ID
  2. Подтвердите удаление, введя "да"
  3. Оценка будет безвозвратно удалена

Просмотр статистики

Функция отображает:

  • Общий средний балл студента по всем предметам
  • Средний балл по каждому предмету отдельно
  • Статистика показывается только по предметам, по которым есть оценки

Поиск по предметам

  1. Выберите предмет из списка по его ID
  2. Система покажет все ваши оценки только по выбранному предмету
  3. Отобразится средний балл по данному предмету

Обработка ошибок

Система включает проверку корректности вводимых данных:

  • Оценки принимаются только в диапазоне от 2 до 5
  • Логины должны быть уникальными
  • Все обязательные поля должны быть заполнены
  • При вводе некорректных числовых значений система выдаст предупреждение

Сохранение данных

Все изменения автоматически сохраняются в файле базы данных students.db. Данные сохраняются после каждой операции и доступны при следующем запуске программы.

Завершение работы

Для выхода из программы:

  • Выберите пункт "0. Выход" в любом меню
  • Или нажмите Ctrl+C для принудительного завершения
Внимание: Все несохранённые данные будут потеряны только в случае принудительного завершения во время операции записи в базу данных.

Устранение неполадок

Проблема: Ошибка "unable to open database file"
Решение: Запустите программу от имени администратора или переместите папку проекта в домашнюю директорию пользователя.

Проблема: Программа не запускается
Решение: Убедитесь, что установлен Python 3.6+ и все файлы проекта находятся в одной папке.

Система готова к использованию и не требует дополнительной настройки.
